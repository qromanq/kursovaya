📁 ОПИСАНИЕ ФАЙЛОВ ПРОЕКТА (Java + Spring Boot + Gradle)

========================================
ФАЙЛЫ ДЛЯ ЗАПУСКА:
========================================

🚀 ЗАПУСК.bat - ГЛАВНЫЙ ФАЙЛ ДЛЯ ЗАПУСКА
   Автоматически устанавливает Java и запускает Spring Boot приложение
   
🔧 gradlew.bat - Gradle Wrapper для Windows
   Автоматически скачивает и использует правильную версию Gradle
   
🔍 test-system.bat - Проверка системы (если есть)
   Тестирует готовность системы к запуску

========================================
ДОКУМЕНТАЦИЯ:
========================================

📖 README.md - Основная документация проекта
📋 ЗАПУСК.txt - Подробные инструкции по запуску
📄 О_ФАЙЛАХ.txt - Этот файл с описанием

========================================
ОСНОВНЫЕ ФАЙЛЫ ПРОЕКТА:
========================================

📦 build.gradle - Конфигурация сборки Gradle
📦 settings.gradle - Настройки проекта Gradle
🐳 Dockerfile - Конфигурация Docker
🚫 .gitignore - Исключения для Git

========================================
ИСХОДНЫЙ КОД JAVA:
========================================

📁 src/main/java/com/inventory/ - Основной код
   📄 InventoryApplication.java - Главный класс Spring Boot
   📁 model/ - Модели данных (JPA Entity)
      📄 Product.java - Модель товара
   📁 repository/ - Репозитории (Spring Data JPA)
      📄 ProductRepository.java - Репозиторий товаров
   📁 service/ - Бизнес-логика
      📄 ProductService.java - Сервис товаров
   📁 controller/ - Контроллеры (REST API и Web)
      📄 ProductController.java - REST API
      📄 WebController.java - Web страницы

📁 src/main/resources/ - Ресурсы приложения
   📄 application.properties - Конфигурация Spring Boot
   📄 data.sql - Тестовые данные для H2
   📁 templates/ - Thymeleaf шаблоны
      📄 index.html - Главная страница
      📄 analytics.html - Страница аналитики
      📄 reports.html - Страница отчетов
      📄 inventory.html - Страница управления запасами

📁 src/test/java/ - Модульные тесты
   📄 InventoryApplicationTests.java - Основные тесты
   📄 ProductControllerTest.java - Тесты REST API

📁 gradle/wrapper/ - Файлы Gradle Wrapper
   📄 gradle-wrapper.properties - Конфигурация Gradle

📁 .github/workflows/ - CI/CD конфигурация
   📄 ci.yml - GitHub Actions pipeline для Java

========================================
АВТОМАТИЧЕСКИ СОЗДАВАЕМЫЕ:
========================================

📁 build/ - Результаты сборки Gradle (создается автоматически)
📁 .gradle/ - Кэш Gradle (создается автоматически)
📁 data/ - База данных H2 (создается автоматически)
📄 *.db - Файлы базы данных H2 (создаются автоматически)

========================================
КАК ИСПОЛЬЗОВАТЬ:
========================================

1. Для запуска: дважды кликните ЗАПУСК.bat
2. Для сборки: gradlew.bat build
3. Для тестов: gradlew.bat test
4. Для изучения: читайте README.md
5. При проблемах: смотрите ЗАПУСК.txt

========================================
ТЕХНОЛОГИЧЕСКИЙ СТЕК:
========================================

- Java 17+ (автоматически устанавливается)
- Spring Boot 3.2 (веб-фреймворк)
- Spring Data JPA (работа с БД)
- H2 Database (встроенная БД)
- Thymeleaf (шаблонизатор)
- Gradle 8.5 (система сборки)
- JUnit 5 (тестирование)

Все настроено для работы "из коробки" на Windows 10/11.
